# firejail

Ansible Role para instalar o [firejail](https://github.com/netblue30/firejail/).

O firejail é compilado do código-fonte.

## Variáveis

- `firejail_version`: Versão do firejail a ser instalada.

## Tags

- `firejail`: Executa todo o role.

## Compatibilidade

- Debian Buster
- Debian Bullseye
